문제점은 이렇게 반응형 웹으로 잘 구성해 뒀는데, pynecone에서 제공하는 호스팅 서비스 (pc deploy)가 아직 준비 중이었다!
그래서 pc export로 html로는 내보낼 수가 있는데 반응형 컴포넌트로는 변환을 하지 못해서(백엔드 파트는 그대로 python 코드로 만든다)
